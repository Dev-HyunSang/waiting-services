# Wating Serverices
> 인기 있는 음식점을 가면 웨이팅은 필수입니다. 웨이팅의 단점은 자신을 부를 때까지 가게 앞(안)에서 기달려야 한다는 단점이 있습니다. 특히 무더운 여름이라면 더위 속에서 맛있는 음식을 먹기 위해서 기달려야 합나다. 그런 단점들을 서비스로 보안하고자 합니다.
